ABSTRACT

Introducing a visual try-on feature (Augmented Reality) to an e-commerce website provides an innovative way for customers to engage with products before making a purchase. By leveraging this feature, users can virtually experience how a specific furniture item would look or fit in their room, ultimately enhancing their shopping experience.

This technology has the potential to increase customer satisfaction and reduce return rates by allowing individuals to make more informed decisions when buying furniture or accessories online. En'Decore is a web application designed using HTML, CSS, JavaScript, and Firebase that features a completely modern, luxurious interface with dynamic rendering capabilities and robust e-commerce features such as a real-time shopping cart, user authentication, and order management.

1. INTRODUCTION

1.1 Company / Brand Profile

En'Decore — Maison of Furniture is a conceptual luxury furniture brand developed as part of this MCA project. The name "En'Decore" is derived from the word "décor," symbolizing elegance and distinction in home furnishing. The brand philosophy is inspired by world-renowned luxury houses such as Hermès, Louis Vuitton, and Prada — emphasizing craftsmanship, timeless design, and premium user experience.

The brand positions itself as a high-end, AR-enabled online furniture store based out of Chennai, Tamil Nadu. Its product catalog includes curated armchairs, sofas, couches, tables, and exclusive designer pieces — each presented with luxurious imagery and detailed information. En'Decore aspires to bridge the gap between physical furniture showrooms and online shopping through immersive Augmented Reality visualization.

1.2 About the Project

The En'Decore AR-Based E-Commerce Platform is a web application that enables users to browse, explore, and purchase luxury furniture online. Its key differentiator is the integration of Augmented Reality (AR), allowing customers to virtually visualize how furniture pieces would look within their own living spaces using their device's camera.

The platform is built using a pure frontend stack — HTML5, CSS3, and Vanilla JavaScript — with Google Firebase powering the backend services including user authentication, real-time database (Firestore), and session management. The design follows a luxury aesthetic with gold-accented color palettes, elegant typography (Cormorant Garamond, Montserrat), smooth micro-animations, and a grain-texture overlay for a premium feel.

Key functionalities include user registration and login, a dynamic product catalog with category filtering and sorting, individual product detail pages with AR "View Piece" integration, a real-time shopping cart with quantity management and GST calculation, order placement with shipping details storage, a contact form connected to Firestore, and a responsive layout across all devices.

1.3 Objectives of the Project

  • Enhanced Product Visualization: Enable users to see how furniture fits in their actual space using AR technology, reducing purchase hesitation.
  • Improved User Experience: Deliver a luxurious, intuitive, and responsive interface that rivals premium brand websites.
  • Increased Sales & Conversions: By allowing virtual try-ons, reduce product return rates and increase buyer confidence.
  • Cross-Platform Accessibility: Ensure the platform works seamlessly across desktops, tablets, and mobile browsers.
  • Real-Time Data Management: Leverage Firebase for instant user authentication, cart synchronization, and order tracking.
  • Solving Real-World Problems: Address the fundamental limitation of online furniture shopping — the inability to visualize furniture at scale in one's own home.

2.3 About the Software / Technologies Used

2.3.1 HTML5

HTML5 (HyperText Markup Language version 5) is the standard markup language used for structuring content on the web. In En'Decore, HTML5 provides semantic elements for the page structure, forms for user input (login, registration, contact, checkout), and the foundational layout for all pages.

2.3.2 CSS3

CSS3 (Cascading Style Sheets Level 3) is used for styling and visual presentation. The project uses a comprehensive custom design system with CSS custom properties (variables), Grid and Flexbox layouts, animations and transitions, glassmorphism effects, and responsive media queries. A luxury design palette featuring ivory, gold, charcoal, and cream tones is implemented entirely in vanilla CSS.

2.3.3 JavaScript (Vanilla ES6+)

JavaScript powers all client-side interactivity including dynamic product rendering, event-driven cart management, Firebase SDK interactions for authentication and database operations, AR modal controls, toast notifications, and DOM manipulation. No external JavaScript frameworks or libraries are used — the application relies on pure vanilla JS for maximum performance.

2.3.4 Google Firebase

Firebase is a Backend-as-a-Service (BaaS) platform by Google that provides: Firebase Authentication for email/password-based user registration and login; Cloud Firestore as a NoSQL real-time database for storing user profiles, shopping carts, orders, and contact form messages. Firebase SDK v9 (compat mode) is used for client-side integration.

2.3.5 AR.js

AR.js is an open-source, lightweight JavaScript library for web-based Augmented Reality. It enables marker-based and markerless AR experiences directly in the browser without requiring native app installation. In En'Decore, AR.js is used to project furniture 3D models into the user's camera feed for virtual try-on.

3. SYSTEM ANALYSIS

3.1 Feasibility Study

3.1.1 Technical Feasibility

The project is technically feasible as it uses well-established web technologies (HTML, CSS, JavaScript) that are universally supported across modern browsers. Firebase provides a reliable and scalable backend without the need for dedicated server infrastructure. AR.js enables browser-based augmented reality without requiring users to download additional software. All technologies used are open-source or free-tier, making them accessible for development and deployment.

3.1.2 Economic Feasibility

The project is economically viable as it uses entirely free and open-source technologies. Firebase offers a generous free tier (Spark Plan) sufficient for development and moderate usage. No paid licenses, hosting server purchases, or third-party subscriptions are required. The only cost is development time and effort.

3.1.3 Operational Feasibility

The system is operationally feasible as it is designed with a user-friendly interface that requires no technical training. The AR feature is accessible via standard device cameras. The platform can be operated by both end-users (customers) and administrators with minimal onboarding. The responsive design ensures accessibility across different device types.

3.2 Existing System

In the existing scenario of online furniture shopping, most e-commerce platforms offer static product images and basic descriptions. Customers face several challenges:

  • Inability to visualize how furniture would actually look in their living space.
  • Size and proportion misjudgments leading to high return rates.
  • Lack of immersive product interaction — users rely solely on 2D photos.
  • Generic website designs that do not convey luxury or brand distinction.
  • Complex checkout processes with poor user experience.

Drawbacks of the Existing System:

  • No AR or virtual try-on capability in most furniture websites.
  • Poor visual representation leads to customer dissatisfaction.
  • High product return rates (estimated 20-30% in online furniture).
  • Customers must visit physical showrooms to verify product suitability.

3.3 Proposed System

The proposed En'Decore system addresses the limitations of the existing system by offering:

  • AR-Based Virtual Try-On: Users can view furniture in their own room through their device camera before purchasing.
  • Luxury UI/UX Design: A premium interface with gold-accented design, elegant typography, smooth animations, and grain texture overlays that elevate the shopping experience.
  • Real-Time Cart & Order Management: Firebase-powered shopping cart with live synchronization, GST calculation (18%), and seamless checkout with delivery details.
  • User Authentication: Secure email/password registration and login using Firebase Authentication, with guest-cart to user-cart migration on signup.
  • Dynamic Product Rendering: JavaScript-driven product catalogs with category filtering (Armchairs, Sofas, Tables) and sorting options (price, name).
  • Contact Form Integration: Customer messages stored directly in Firebase Firestore for easy management.
  • Responsive Design: Full compatibility across desktops, tablets, and mobile devices.

Advantages of the Proposed System:

  • Reduces product returns by enabling informed purchasing decisions.
  • Increases customer engagement through immersive AR experiences.
  • No server maintenance required (Firebase BaaS handles infrastructure).
  • Fast page loads with lightweight vanilla JavaScript architecture.
  • Scalable to accommodate additional product categories and features.

5. SYSTEM DESCRIPTION (Modules)

The En'Decore platform consists of six key functional modules:

5.1 Home Page Module (index.html)

The home page serves as the primary entry point to the platform and includes:

  • Hero Section: A full-viewport split-screen layout with brand tagline ("Where Craft Meets Vision"), call-to-action buttons, and a high-quality luxury interior image with a Ken Burns zoom effect on hover.
  • Category Strip: A three-column image grid (Armchairs, Sofas, Tables) with hover animations linking to the products page.
  • Featured Products: Dynamically rendered product cards showing Signature and curated pieces with images, ratings, prices, and AR quick-view buttons.
  • Latest Collection: A second product row highlighting new arrivals with "New" badges.
  • Feature Banner: An exclusive product spotlight (Luxury Grey Sofa) with a dark-themed full-width section.
  • Testimonials: Client review cards with star ratings and author initials.
  • Footer: Navigation links, social media icons, and brand tagline.

5.2 Product Catalog Module (products.html)

The collection page displays all available furniture products with:

  • Category Tabs: Filter buttons for All, Armchairs, Sofas, and Tables.
  • Sort Dropdown: Options for Default Sorting, Price Low-to-High, Price High-to-Low, and Name A–Z.
  • Product Cards: Each card displays the product image, category label, name, star rating, price, "View Piece" overlay on hover, product badge (Signature/New/Exclusive), and AR quick-view button.
  • All 13 products are rendered dynamically using JavaScript arrays and template literals.

5.3 Product Detail Module (pd_*.html)

Each product has a dedicated detail page featuring:

  • Large product images with multiple views.
  • Product title, description, category, and price.
  • "View in AR" Button: Launches the AR modal for virtual placement.
  • "Add to Cart" Button: Adds the product to the user's Firebase cart.
  • Related product suggestions.

5.4 Shopping Cart Module (cart.html)

The cart page provides a complete shopping experience:

  • Cart Table: Lists all added products with image, name, quantity controls (+/−), and line total.
  • Order Summary: Displays Subtotal, GST at 18%, Complimentary Shipping, and Grand Total.
  • Empty Cart State: Displays a styled empty state with a link to explore the collection.
  • Checkout Flow: Requires sign-in → reveals shipping form → captures delivery details → places order to Firestore.

5.5 Account Module (account.html)

The account page handles user authentication and profile management:

  • Login Form: Email and password sign-in using Firebase Authentication.
  • Registration Form: Creates new user with name, email, and password; stores profile in Firestore.
  • Dashboard View: Shown after login — displays welcome message, quick links to Cart, Order History, and user email.
  • Order History: Fetches and displays all past orders from Firestore with order ID, items, status, and total.
  • Sign Out: Logs the user out and returns to the authentication view.

5.6 Additional Pages

  • About / Maison Page (about.html): Brand story, company philosophy, and values.
  • Contact Page (contactus.html): Contact form that saves messages to Firebase Firestore's 'messages' collection; also includes phone and email info.
  • Blog / Journal Page (blogs.html): Grid layout for articles and content related to furniture and interior design.
  • FAQ Page (faq2.html): Frequently asked questions about the platform, AR feature, shipping, and policies.

6. TESTING AND IMPLEMENTATION

6.1 Testing Strategy

Testing was performed at multiple levels to ensure the quality, reliability, and usability of the En'Decore platform.

6.1.1 Unit Testing

Individual functions and modules were tested independently:

  • Product rendering functions were verified for correct card generation.
  • Star rating function was tested for whole, half, and empty star outputs.
  • Cart arithmetic was validated: subtotal calculation, GST (18%), and total computation.
  • Firebase authentication functions were tested for successful login, registration, and error handling.

6.1.2 Integration Testing

Combined modules were tested for correct data flow:

  • Adding products to cart and verifying reflection on the cart page.
  • User login followed by cart access and order placement.
  • Contact form submission and verification in Firestore console.
  • Guest-cart to authenticated-cart migration on user registration.

The En'Decore platform was implemented in the following phases:

  2. Phase 2 — Page Structure: Built all HTML pages with semantic structure, navigation, and footer components.

7. CONCLUSION AND FUTURE SCOPE

7.1 Conclusion

The En'Decore — AR Based E-Commerce Platform has been successfully designed, developed, and tested as part of this MCA project. The platform demonstrates the effective integration of modern web technologies (HTML5, CSS3, JavaScript) with cloud services (Firebase) and cutting-edge Augmented Reality (AR.js) to create a premium online furniture shopping experience.

Key achievements of the project include:

  • A fully functional AR-enabled e-commerce website with luxury aesthetics.
  • Complete user authentication and session management using Firebase.
  • Real-time shopping cart with dynamic pricing, GST calculation, and order placement.
  • A responsive, cross-browser design that works across desktops, tablets, and mobile devices.
  • A custom vanilla JavaScript architecture with no external framework dependencies, resulting in fast load times and lightweight performance.

The project addresses a genuine gap in the online furniture market by enabling users to visualize products in their own space before purchasing, thus reducing return rates and improving customer satisfaction.

7.2 Future Scope

  • Expanded 3D Product Catalog: Continuously update and expand the collection of AR-ready 3D furniture models for a richer visualization experience.
  • Color & Material Customization: Allow users to change furniture colors, fabrics, and finishes directly within the AR environment.
  • Real-Time Collaboration: Enable multiple users to view and interact with the same AR environment simultaneously for joint decision-making.
  • AI-Powered Recommendations: Implement machine learning algorithms to suggest furniture based on user preferences, browsing history, and room dimensions.
  • Payment Gateway Integration: Integrate real payment systems (Razorpay, Stripe, UPI) for actual transaction processing.
  • Admin Dashboard: Build an administrative panel for managing products, orders, inventory, and customer queries.
  • Push Notifications: Implement Firebase Cloud Messaging for order status updates, promotions, and new arrival alerts.
  • Progressive Web App (PWA): Convert the website to a PWA for offline access, faster loads, and installable mobile experience.
  • Multi-Language Support: Add language localization for wider audience reach.

ABSTRACT
This project presents the design and development of a Virtual Try Feature
integrated into an e-commerce website to enhance online shopping through
Augmented Reality (AR). The system enables users to visualize selected products
in real-world environments before purchase, providing an interactive and
immersive shopping experience. The prototype supports multiple products and
allows real-time visualization, 3D rotation, and basic customization features to
help users make informed buying decisions.

The application is developed using Unity and C# for AR-based product
visualization, while the web interface is built using HTML, CSS, and JavaScript to
ensure a responsive and user-friendly experience. The system integrates AR/3D
components with REST-style backend services to manage product data and user
interactions efficiently. Version control using Git was implemented to manage
development iterations and team collaboration.

The proposed system aims to bridge the gap between digital product representation
and physical experience, thereby improving customer engagement, increasing
purchase confidence, and reducing product return rates. This solution demonstrates
the potential of AR technology in transforming traditional e-commerce platforms
into interactive and intelligent shopping environments.
